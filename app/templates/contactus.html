Thank you for using our website! If you have found a bug, problem or want to pass on a compliment to the creators please contact: Peng Cao at sylarcp@udel.edu

Alternitively, you can create an issue on github by clicking on the following link: https://github.com/sylarcp/anita/issues 
Any details you can provide about the problem or bug will help contribute to our team creating a speedy fix.

To create an issue on github simply follow these five steps:
1. Make sure you are signed into github, if you have not created an account you should do so now.
2. Click on the link mentioned above which will bring you to the issues page for the website.
3. There should be a green button that says "New issue", click on it.
4. Create a write up of the problem with any relevent information you can think of.
5. Once you are finished, hit the green "Submit new issue" button and your work is done.
